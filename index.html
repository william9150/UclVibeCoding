Vibe Coding 是如何進行的？

整個過程就像與一位專業開發者對話，您只需要用自然語言描述想法，AI 就會為您完成繁瑣的程式碼工作。這是一個簡單四步驟的循環：

1. 環境設定

安裝基本工具：VS Code 編輯器、Node.js 環境與 Gemini CLI。

2. 描述需求

開始與 Gemini CLI 對話，用中文或英文描述您想要的網頁功能或外觀。

3. 檢視成果

Gemini 會快速生成對應的 HTML 程式碼，您可以直接在 VS Code 中預覽結果。

4. 持續優化

不滿意嗎？繼續對話，提出修改指令，例如「按鈕改大一點」或「換個顏色」。
